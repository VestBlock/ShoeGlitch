# Event Taxonomy

## Stable event names
- `page_view`: route was viewed
- `cta_click`: a tracked CTA was clicked
- `lead_submit`: a lead form was submitted
- `operator_interest`: an operator-intent flow was started
- `watchlist_save`: a user saved a sneaker to watch
- `booking_start`: a booking flow was started
- `booking_complete`: a booking was completed

## Required fields
- `routePath`
- `eventName`

## Preferred metadata
- `pageTitle`
- `href`
- `templateFamily`
- `service`
- `city`
- `slug`
- `ctaContext`

## CTA label guidance
- Use the visible CTA label or a close normalized label.
- Keep labels human-readable.
- Avoid internal phrases like `primary_cta` or `variant_a`.
